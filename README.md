DESCRIPCION DEL PROYECTO
Este proyecto es un panel de control de clientes desarrollado con React y Vite. La aplicación permite iniciar sesión como administrador, navegar entre distintas pantallas protegidas y consultar una lista de clientes obtenida desde FakeStoreAPI. También se implementó una ficha detallada para cada cliente, donde se muestran sus datos personales, dirección y credenciales. Según el sector del administrador, la aplicación permite solo visualizar la información o realizar acciones como marcar un cliente como eliminado. Para mejorar la interfaz se utilizaron componentes de Material UI, como tablas, botones, alertas, modales y tarjetas.
